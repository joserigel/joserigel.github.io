Sleek Weather App

This is a Weather App that uses Open-Meteo to fetch forecast weather data around the world. The frontend uses React.js with Express.js as its static file server. The backend uses Express.js, to connect to the MySQL Database. Deployment is managed using Azure with Docker.

The goal of this Project is to exercise my frontend skill as well as deployment of a project.

Technologies used:

  • React.js
  • Express.js
  • MySQL
  • Open-Meteo API
  • Docker
